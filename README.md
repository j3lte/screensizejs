screensizejs
============

Just a small script you can insert to view your screensize in EM

## Getting Started

Insert the script in the page

```
<script type="text/javascript" src="./screensize.js"></script>
```

## Usage

Make sure you reset the font-size on the body tag

```
<style type="text/css">
  body {
    font-size:16px;
  }
</style>
```
